# Task Smash

A simple task manager web app built using Flask.

## Live Demo

https://task-smash-bnk6.onrender.com

## Features

* Add tasks
* Update tasks
* Delete tasks
* Stores tasks using SQLite

## Tech Stack

* Python
* Flask
* Flask-SQLAlchemy
* HTML/CSS
* SCSS
* SQLite
* Gunicorn

## Run Locally

Clone the repository:

```
git clone https://github.com/your-username/task-smash.git
cd task-smash
```

Create a virtual environment:

```
python -m venv .venv
source .venv/bin/activate
```

Install dependencies:

```
pip install -r requirements.txt
```

Run the app:

```
python app.py
```
